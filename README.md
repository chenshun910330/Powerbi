🛠️ 核心技術與開發流程

數據整合與處理：

彙整 2011–2020 官方統計數據（人口結構、建物樓層、家庭收入與支出）。

執行資料清洗（Data Cleaning）與多維度關聯建模。

開發工具：Microsoft Power BI。

動態交互設計：

建構高度連動性（Cross-filtering）界面。

實現全頁面即時同步更新與探索式分析。

✨ 技術亮點：六維動態分析模型
透過「動態泡泡圖」呈現高密度資訊量：

X 軸：數值維度一

Y 軸：數值維度二

氣泡大小：數值維度三

氣泡左/右：數值維度四、五

播放軸：動態時間維度（10 年趨勢演變與異常值識別）
