# 2011–2020 社會經濟與人口趨勢互動式視覺化分析
### Socio-Economic & Demographic Trend Interactive Visualization (2011–2020)

## 🏆 作品成就
* **校內視覺化分析競賽：第二名**
  * 本作品開發於大三「視覺化分析」課程，並在校內競賽中獲得高度評價。

---

## 🛠️ 核心技術與開發流程

### 1. 數據整合與預處理 (Data Integration & Cleaning)
* **資料來源**：彙整 2011 年至 2020 年之官方統計數據。
* **涵蓋範疇**：人口結構、建物樓層分佈、家庭收入及各項支出等多元 Excel 來源。
* **技術執行**：進行資料清洗 (Data Cleaning) 以確保資料品質，並建立多維度資料關聯建模。

### 2. 動態交互設計 (Interactive Design)
* **開發工具**：Microsoft Power BI。
* **設計邏輯**：建構具備高度連動性 (Cross-filtering) 的互動式介面。
* **使用者體驗**：使用者可選取或縮放特定資料項，全頁面關聯圖表將即時同步更新，實現直觀的「探索式分析」。

---

## ✨ 技術亮點：六維動態分析模型
本專案之核心競爭力在於**六維度動態泡泡圖**，透過單一視覺化組件呈現極高密度的資訊量，協助分析者識別趨勢與異常值：

| 維度 | 定義 / 功能描述 |
| :--- | :--- |
| **X 軸** | 數值維度一 |
| **Y 軸** | 數值維度二 |
| **氣泡大小** | 數值維度三 |
| **氣泡左側** | 數值維度四 |
| **氣泡右側** | 數值維度五 |
| **播放軸** | **動態時間維度**（呈現 10 年間之演變路徑） |

> **設計價值**：藉由時間軸的播放功能，能將複雜的數據流轉化為具備洞察力的視覺敘事，精準掌握長期趨勢。
